## 🎰 任務抽卡機｜多巴胺回饋模組細化 v1.0

**關鍵詞**：隨機性、微任務、即時成功感、趣味驅動

### 模組功能定位

| 項目 | 說明 |
| --- | --- |
| 模組名稱 | 任務抽卡機（Micro Mission Gacha） |
| 功能類型 | 任務生成＋回饋系統 |
| 使用對象 | ADHD / HSP、學習動力低、起步困難者 |
| 任務範疇 | 可選領域（語言 / 寫作 / 運動等） |
| 使用頻率 | 每日 1–3 次，搭配誘餌卡更佳 |

### 核心機制設計

1. 🎯 任務池管理
   - 依主題分類；任務需 5–10 分鐘可完成、操作明確、成果可感
   - 支援預設池＋使用者自定義（可標記「我想抽到這個！」）
2. 🎲 抽卡機制
   - 每次抽取一張任務卡：標題、時間標籤、抽換限制、完成按鈕＋回饋
3. 🧸 回饋設計
   - 完成即觸發療癒回饋動畫；連續完成可得稱號（示例）
4. 🧠 智能卡池（AI 擴展位）
   - 目前使用文字庫隨機；未來可接 GPT 依使用者目標生成

### UI/UX 簡化示意

- 扭蛋式抽卡；開盒動畫；「我不要這張」有趣表達（貓咪把卡叼走）

## 本 MVP 與原型的對應

- 已實作
  - 奇蹟日記：編輯/提交/貼紙回饋；Sticker Wall 分組顯示
  - 任務抽卡：任務池、類型、用戶任務與管理（抽卡頁、限制/代幣留位）
  - 誘餌卡＋情緒：心情選擇、誘餌建議、自定義心情與別名
  - 回饋中樞：Reward Center（貼紙＋紙屑），事件解耦（CustomEvent）
- 已留擴展位
  - AI 任務拆分、AI 日終總結（模板註冊、local provider）
  - 指標聚合與圖譜（metrics store、analytics adapter）
